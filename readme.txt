Easily force single page cache without waiting when using Swift Performance.

== Description ==

Adds a button to the bottom of front-end pages so you can force cache creation of any site page without waiting for it to build manually.

<strong>ONLY WORKS WITH SWIFT PERFORMANCE caching plugin!</strong>
